# Dengue-Chikungunya-Oropouche-prediction
Predictive models (Random Forest/Ranger) for differential diagnostic screening of Dengue, Oropouche, and Chikungunya. Developed in R as part of clinical research to support decision-making in arboviruses. Includes ready-to-use .RData files. Go to https://pedrobrasil.shinyapps.io/INDWELL/ for living example.
